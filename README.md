## 📌 Overview

ITALK is a cloud-based real-time communication platform designed for universities to enable seamless interaction between students and professors. It introduces a *classroom-based communication model* combined with instant messaging using WebSockets.

The system is deployed using a multi-cloud architecture, ensuring scalability, performance, and high availability.

---

## 🌟 Features

- 💬 Real-time messaging using Socket.IO  
- 🏫 Classroom-based communication system  
- 📢 Broadcast messaging for announcements  
- 🔐 User authentication  
- ☁️ Multi-cloud deployment (Render + Vercel + Railway)  
- ⚡ Low-latency communication without page refresh  

---

## 🛠 Tech Stack

**Frontend:**  
- HTML, CSS, JavaScript (Vercel)

**Backend:**  
- Node.js, Express.js (Render)

**Database:**  
- MySQL (Railway Cloud)

**Real-Time Communication:**  
- Socket.IO (WebSockets)

---

## 🏗 System Architecture

The system follows a layered cloud architecture:

- Frontend Layer (Vercel CDN)  
- Backend Layer (Node.js + Express on Render)  
- Real-Time Layer (Socket.IO)  
- Data Layer (Railway MySQL)

## ⚡ How It Works

1. User registers or logs in  
2. User joins a classroom  
3. Messages are sent via WebSocket connection  
4. Server broadcasts messages to classroom members  
5. Data is stored in MySQL database
